## 获取脚本所在路径
```shell
export SCRIPT_HOME=$(dirname $(readlink -f $0))
```

## 解压压缩
```shell
# tar.gz
tar -zxvf filename.tar.gz 
tar -zcvf filename.tar.gz target

# tar
tar -xvf filename.tar 
tar -cvf filename.tar target

# zip
zip -r ./*
unzip filename.zip 
```
